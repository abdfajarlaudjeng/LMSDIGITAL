# LMSDIGITAL
🌟 Cara Kerja Sincronisasi & Hak Akses Fitur:
Guru Workspace: Memiliki 3 formulir panel kontrol khusus di sisi kiri untuk merilis modul materi, membuat tautan ruang kelas virtual (Zoom/Meet), membagikan kuis/tugas, serta meng-update angka capaian progres siswa.

Siswa Workspace: Mengakses panel kanan secara otomatis. Materi, tautan video conference kelas virtual, tugas, kuis, dan tabel pelacakan progres belajarnya tersinkronisasi murni berdasarkan nama mereka serta mata pelajaran terkait. Siswa juga dapat berdiskusi aktif di forum kelas.

Orang Tua Workspace: Pada pangkalan data User, baris akun Orang Tua wajib diisi kolom target_siswa (misal: Budi Santoso). Ketika login, sistem akan menyaring data (filtering) secara otomatis sehingga Orang Tua hanya bisa melihat tabel pelacakan capaian belajar khusus untuk nama anak mereka demi menjaga privasi.

Pengawas Workspace: Akun pengawas didesain untuk monitoring makro (keseluruhan sekolah). Pengawas akan melihat seluruh data performa nilai siswa dari semua kelas tanpa filter apa pun untuk kebutuhan evaluasi kurikulum dan penjaminan mutu sekolah.
